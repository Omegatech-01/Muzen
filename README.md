# 🎵 Muzen — Premium Personal Music Workspace

Muzen is a lightweight, responsive, and secure music companion designed for instant creative expression. Describe a vibe, a feeling, or a scene, and Muzen handles the orchestration, lyrical writing, and composition on the fly. 

Designed for high-fidelity audio playback and complete offline persistence.

---

## 🌟 Features

- **Double Engines**: Supports **Muzen-1.0** for catchy hooks and vocal compositions, and **Muzen-2.0** with style parameter selectors (genres, moods, vocal profiles, and tempo control) + advanced Custom/Manual lyrics writing.
- **Micro-Interactions & Motion**: Driven by GSAP (GreenSock Animation Platform) and fluid CSS transitions for state shifts, splash stages, routing navigations, and floating toast completions.
- **Synced Lyrical Engine**: High-fidelity scrolling karaoke lyrics perfectly synchronized with the playing stream.
- **Local Library & Persistence**: Built-in client-side serialization to persist saved tracks, playback queues, and preferences with no sign-up or credit card.
- **Secure Auto-Registration**: Device-level automated cryptographic unique identifiers for server interactions, safe from crawlers and automated bots.
- **Handcrafted SVGs**: Built entirely with responsive, lightweight, crisp vector artwork with no bulky external font packages.
- **Quick Share & Portability**: Native social quick-links to share track pages directly to TikTok, Facebook, and GitHub, or download high-quality MP3 tracks directly.

## 🚀 Getting Started

### Prerequisites

You need Node.js (v18+) and npm installed on your machine.

### Installation

1. Clone or fork this repository:
   ```bash
   git clone https://github.com/Omegatech-01/Muzen.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create your environment configuration:
   ```bash
   cp .env.example .env
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

6. Start production server:
   ```bash
   npm run start
   ```
